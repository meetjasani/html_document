Requirements & Compatibility

To run WordPress your host just needs a couple of things:

  • PHP version 7.3 or greater
  • MySQL version 5.6 or greater

Our theme is compatible with the following browsers

  • Chrome
  • Firefox
  • Safari
  • Microsoft Edge

Theme Installation

To install using WordPress, simply follow the steps outlined below:

Step 1 : Login to the WordPress Dashboard.

Step 2 : Appearance -> Themes -> Add New

Step 3 : Upload Basis Theme -> select file

Step 4 : After downloading the theme file, you can install it by selecting the file and then clicking on the "Install Now" button.

Step 5 : Then Upload Child theme and click "Install Now" button

Step 6 : When the theme is installed successfully then activate the "child theme"

One Click Demo Import

We offer sample data that you can use to make your store resemble our demo.

Delete Extra Pages

Step 1 : Go To Pages -> All Pages

Step 2 : Select the Title option

Step 3 : Click on bulk actions -> Choose Move to Trash then "Apply"

Step 4 : Select to the "Trash" option

Step 5 : choose the Delete permanently then "Apply"

To import the sample data please ad here to the following steps:

We offer sample data that you can use to make your store resemble our demo.

Check server settings and activate required plugins before proceeding with the next step. See details below.

Given the below details have to include in ".htaccess" File

php_value upload_max_filesize 600M
php_value post_max_size 600M
php_value memory_limit 256M
php_value max_execution_time 300
php_value max_input_time 300

Step 1 : Go to Appearance -> Select Import Demo Data

Step 2 : After Click On Import Demo Data, You should select the below-highlighted files with mentioned these three extensions that are provided in the "Sample Data" folder then click on "Continue & Import" Button

Now you can see all data import complete!

Note : When you do a one-click demo import on localhost and whenever it will complete then you should have to follow below given steps

Open the sample data folder and select the widgets file and open it then copy all codes given in the file.

GO to the tool and open widgets importer & exporter.

Click on copy & paste link and it will show a white box then you have to paste the all code here that we copied from step 1. After this click on the Import Widgets button.
